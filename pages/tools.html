Tools

This is my list of tools.

security

  • nikto an Open Source server scanner
  • dirbuster a multi threaded java application designed to brute force directories and files names on web/application servers
  • nmap an open source utility for network discovery and security auditing
  • sqlmap an open source penetration testing tool that automates the process of detecting and exploiting SQL injection flaws
  • sqlninja a SQL Server injection & takeover tool
  • wireshark the world's foremost network protocol analyzer
  • burp suite an integrated platform for performing security testing of web applications
  • hydra very fast network logon cracker which support many different services
  • john the ripper a fast password cracker
  • hashcat an advanced password recovery
  • mimicatz a little tool to play with Windows security
  • wpscan a black box WordPress vulnerability scanner
  • radare2 an unix-like reverse engineering framework and commandline tools
  • volatility an advanced memory forensics framework
  • net-creds a pcap sniffer
  • crunch a password list generator
  • cewl a password list generator from a webpage
  • searsploit allow to searh for an exploit localy

Burp extensions

  • Wsdler "This extension takes a WSDL request, parses out the operations that are associated with the targeted web service, and generates SOAP requests that can then be sent to the SOAP endpoints."
  • Hackvertor replace the decoder

Firefox

  • foxyproxy an advanced proxy management tool

tools for linux and i3

  • i3 a tiling window manager, see my config on github
  • vim a highly configurable text editor, see my config on github
  • neovim a highly configurable text editor, see my config on github
  • ranger A VIM-inspired filemanager for the console, see my config on github
  • git a distributed version control system
  • tig an ncurses-based text-mode interface for git
  • htop an interactive process viewer for Linux
  • ncdu a disk usage analyzer with an ncurses interface
  • pacgraph draws a pretty picture of your installed packages
  • evince a document viewer for multiple document formats
  • sxiv Simple X Image Viewer
  • cmus is a small, fast and powerful console music player
  • vlc a free and open source cross-platform multimedia player
  • nemo a File browser
  • krita a FREE sketching and painting program
  • darktable an open source photography workflow application and RAW developer.
